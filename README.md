scrimfinder
=======

A platform for organizing and scheduling Team Fortress 2 scrims.

Requirements
=======
	python 2.x

	pip install sqlalchemy
	pip install flask-sqlalchemy
	pip install flask
	pip install requests
	pip install flask-openid
	pip install wtforms
